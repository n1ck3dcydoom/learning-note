# 前序和后序遍历树问题：

## 1 若一棵二叉树的前序遍历序列为 a, e, b, d, c，后序遍历序列为 b, c, d, e, a，则根结点的孩子结点是？

答：  
首先根据先序遍历可知：根节点是 `a`

* 假设 `a` 有两个孩子：
由于先序遍历中 `e` 在 `a` 的后面，所以 `e` 只能是 `a` 的左子树下的节点，而后序遍历中 `e` 在 `a` 的前面，所以 `e` 是 `a` 的右子树下的节点，相互矛盾。

* 假设 `a` 只有一个孩子，那么 `e` 就是 `a` 的左孩子

1. 那么有序树T中结点的前序就是二叉树T1中结点的前序
